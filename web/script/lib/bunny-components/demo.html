bunny-components

Material Component pack by codingchili@github only using lit-html.

npm install --save codingchili/bunny-<name>

bunny-progress

Shows a horizontal progress bar with themable and animatable progress.

<style>
    bunny-progress {
        --bunny-progress-transition-timing-function: ease-out;
        --bunny-progress-transition-duration: 0.16s;
        --bunny-progress-height: 8px;
        --bunny-progress-active-color: #ffcc800;
        --bunny-progress-container-color: #484848;
    }
</style>

<bunny-progress value="20" max="100"></bunny-progress>

bunny-tooltip

hover me location="top" location="left" location="right" location="bottom"
A tooltip with support for popup location and complex content.
can target a specific node using target or previous sibling by default. 

<script type="module" src="bunny-tooltip.js"/>

<-- automatically targets previous node in tree. -->
<div id="target">tooltippable</div>
<bunny-tooltip location="top">location="top"</bunny-tooltip>

<-- cannot use previous node, use explicit 'for' to set target. -->
<bunny-tooltip location="bottom" for="target">location="bottom"</bunny-tooltip>
